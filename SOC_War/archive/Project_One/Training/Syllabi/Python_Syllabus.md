# 🐍 Python for Automation & Offensive CLI – Project ONE Syllabus

## 🔹 LEVEL 1 — Terminal Toolkit
- Variables & `input()`
- Strings & f-strings
- Lists & dictionaries
- If-else logic
- Loops: `for`, `while`
- Functions & arguments

## 🔹 LEVEL 2 — File & Data Handling
- `open()`, `read()`, `write()`
- `json` module
- `csv` module
- Try/except: exception handling

## 🔹 LEVEL 3 — Modules & CLI App Structure
- Standard modules: `os`, `sys`, `time`
- Creating custom `.py` modules
- `argparse` for command-line arguments
- Virtual environments (`venv`)

## 🔹 LEVEL 4 — Automation Flows
- `os.system()` / `subprocess`
- Crontab integration
- Auto-backup scripts
- CLI menu workflows

## 🔹 LEVEL 5 — Security Scripting
- `getpass`, `secrets`
- `cryptography` (Fernet)
- `hashlib` for hashes
- Flask (optional web access layer)
