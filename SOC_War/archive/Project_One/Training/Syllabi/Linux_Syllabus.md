# 🐧 Linux Mastery Syllabus – Project ONE CLI Phase

## 📁 Phase 1 – CLI Essentials
- File navigation: `ls`, `cd`, `pwd`, `mv`, `cp`, `rm`
- File types & permissions: `file`, `chmod`, `chown`
- Redirection: `>`, `>>`, `<`, `|`, `tee`
- Viewing: `cat`, `less`, `head`, `tail`

## ⚙️ Phase 2 – Users & System Control
- User mgmt: `adduser`, `passwd`, `usermod`
- Groups: `groupadd`, `/etc/group`
- Processes: `ps`, `top`, `kill`, `jobs`, `fg`, `bg`
- Services: `systemctl`, `cron`, `crontab`

## 📦 Phase 3 – Bash & Parsing
- Bash scripting basics
- `awk`, `sed`, `cut`, `sort`, `uniq`, `wc`
- Scripting `.sh` files with executable bits

## 🌐 Phase 4 – Networking & Privilege
- Net tools: `netstat`, `ss`, `ping`, `nmap`, `nc`
- SSH & SCP
- File security: sudo, permissions, audit logs
- Threat modeling: `/var/log/`, user traces
