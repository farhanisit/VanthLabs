## 🔗 CLI Ritual #013 – Piping: Chaining Command Power

**Date:** 2025-05-16  
**Focus:** `|` (pipe), command chaining

**Commands Practiced:**
- `cut -d ' ' -f1 access.log | sort | uniq`
- `... | uniq -c | sort -nr | head -n 3`
- `... | tee top3_ips.log`

**Use Cases:**
- Log parsing
- Recon chaining
- Tool pipelines

**Vanth Insight:**
One command is powerful. But chained, they become unstoppable.  
This is the bloodstream of the Unix system — pipes turn data into motion.
