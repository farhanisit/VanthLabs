# 🧠 Weekend Utility Ideas – Vanth Forge

## 🔧 1. Vanth TaskForge (Terminal-Based Schedule Setter)
- Python-based CLI app to set and log daily tasks
- Prompts: task name, start time
- Output: clean saved `.json` or `.csv` log
- Future upgrade: view mode, daily reset, priority tagging

## 🌍 2. WorldClock CLI
- Python script to check time in any country/city
- Input: country or city name
- Output: current time (with or without API)

## 💱 3. Currency Converter CLI
- Python script to convert between currencies
- Input: "USD to INR"
- Output: latest rate (mocked or API-driven)
- Future upgrade: track daily exchange logs
